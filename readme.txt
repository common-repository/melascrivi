Manage your orders for the Melascrivi or hotype marketplace.

== Description ==

This plugin include in the Wordpress Dashboard all the services of Melascrivi.com and Hotype.co.uk. Every WP blog owner will be able to ask, review, accept and then publish unique contents on his website. Simple and strongly recommended for SEO needs.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `melascrivi_plugin.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Sign up or register to request new orders
